# CCalc
Compiles on Linux and Windows with MinGW.

## Build
- Clone repository using
```git clone https://github.com/akachronix/CCalc.git```

- Mark script as executable
```chmod +x compile.sh```

- Success.

## Install (Linux)
- Clone repository using
```git clone https://github.com/akachronix/CCalc.git```

- Mark script as executable
```chmod +x install.sh```

- Success.
